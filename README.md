# DOOMU CHAN(づ｡◕‿‿◕｡)づ
a not so versatile doom based engine being made with c and sdl2

## Try it out
Dependencies:
gcc, libsdl2-dev


Installing and running doomu-chan
```sh
git clone https://github.com/bcbro2021/Doomu-chan-Engine.git
cd Doomu-chan-Engine
sh run.sh
```
